# Hermes Skill Quality Rubric

Each category is scored 0 to 3.

## Categories

- Zweckklarheit: 0 absent, 1 vague, 2 usable, 3 precise and scoped.
- Aktivierungskriterien: 0 absent, 1 implied, 2 explicit positive triggers, 3 positive and negative triggers.
- Arbeitsablauf: 0 absent, 1 loose notes, 2 ordered workflow, 3 workflow with checkpoints.
- Tool-Safety: 0 absent, 1 generic caution, 2 concrete limits, 3 explicit stop/approval rules.
- Output-Vertrag: 0 absent, 1 implied output, 2 explicit deliverable, 3 format plus evidence requirements.
- Evalbarkeit: 0 absent, 1 subjective, 2 checkable criteria, 3 runnable or repeatable eval.
- Wartbarkeit: 0 brittle/overlong, 1 hard to maintain, 2 reasonably scoped, 3 concise and easy to update.
